# Calculadora de Hora Extra

Aplicação para calcular hora extra de trabalho.

**Regras de calculo**:
- Acima de 8 horas de trabalho.
- Acrecimo de 55%: as duas primeiras horas de segunda a sexta feira nos dias uteis.
- Acrecimo de 60%: apos as duas primeiras horas de segunda a sexta feira nos dias uteis.
- Acrecimo de 100%: sabados, domingos e feriados.
